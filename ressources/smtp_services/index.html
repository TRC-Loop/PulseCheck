    Email Services and Their SMTP Domains¶

    This table provides an overview of popular email services and their SMTP servers. It is especially helpful for configuring email clients or troubleshooting email sending issues.

    Services and SMTP Domains¶

    Service SMTP Domain Port Encryption Additional Notes
    Gmail smtp.gmail.com 465 / 587 SSL / TLS App password required. (support.google.com)
    Outlook.com smtp-mail.outlook.com 587 STARTTLS IMAP/POP must be enabled in the settings. (support.microsoft.com)
    Yahoo Mail smtp.mail.yahoo.com 465 / 587 SSL / TLS SSL/TLS required. (help.yahoo.com)
    AOL Mail smtp.aol.com 465 SSL SSL required. (help.aol.com)
    Zoho Mail smtp.zoho.com 465 / 587 SSL / TLS SSL/TLS required. (zoho.com)
    ProtonMail smtp.protonmail.com 465 SSL SMTP token required. (proton.me)
    iCloud Mail smtp.mail.me.com 587 STARTTLS App-specific password required. (support.apple.com)
    Mailgun smtp.mailgun.org 587 STARTTLS SMTP username and password required. (documentation.mailgun.com)
    SendGrid smtp.sendgrid.net 587 STARTTLS API key required. (twilio.com)
    FastMail smtp.fastmail.com 465 / 587 SSL / TLS SSL/TLS required. (fastmail.help)
    GMX Mail mail.gmx.com 465 / 587 SSL / TLS SSL/TLS required. (serversmtp.com)
    Yandex Mail smtp.yandex.com 465 / 587 SSL / TLS SSL/TLS required. (yandex.com)
    Office 365 smtp.office365.com 587 STARTTLS Authentication required. (learn.microsoft.com)
    Strato Mail smtp.strato.de 465 SSL / TLS SSL/TLS required. (strato-hosting.co.uk)
    Tutanota No SMTP support - - Web client only. (reddit.com)
    Rackspace Email secure.emailsrvr.com 465 / 587 SSL / TLS Authentication required. (learn.microsoft.com)
    Bluehost Mail mail.smtp.bluehost.com 465 / 587 SSL / TLS Authentication required.
    HostGator Mail mail.hostgator.com 465 / 587 SSL / TLS Authentication required.

    Important Notes¶

    • Authentication: Most email services require authentication to send emails. Make sure you use your full email address (e.g., name@domain.com) and corresponding password.
    • Encryption: Always use SSL/TLS encryption when available to secure your email transmission.
    • Port Numbers:
    • Port 465: Commonly used for SSL connections.
    • Port 587: Commonly used for TLS connections.
    • Tutanota: Tutanota does not support SMTP. You must use their web client or mobile app for email access.
    • App Passwords: Some services, like Gmail and iCloud, require app-specific passwords for authentication in email clients.
